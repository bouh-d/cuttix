# Cuttix

LAN administration and audit toolkit. Scan, control, capture, detect — one tool.

> Work in progress. CLI-first, GUI later.

## Install (dev)

```bash
git clone https://github.com/YOUR_USERNAME/cuttix.git
cd cuttix
make dev
```

## Usage

```bash
sudo cuttix --help
```
